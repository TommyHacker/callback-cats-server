# Getting Started
<hr>
## <i> clone this repo</i>

##### Then in the terminal:

        - cd api
        - npm install
        - npm run dev

##### Then in the browser:
        - http://localhost:3000
<hr>

### Endpoints

    -   localhost:3000/ should show welcome message
    -   localhost:3000/users show all users if authenticated
    -   localhost:3000/users/:id show own authenticated data

#### Server is working!

<hr>
